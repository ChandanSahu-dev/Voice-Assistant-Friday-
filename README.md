# Voice Assitant (Friday)
FRIDAY is a smart voice assistant built using React that allows users to interact with the application through voice commands.Inspired by modern AI assitants, Friday leverages speech recognition to understand user input and integrates external APIs like Google and Wikipedia to provide accurate, fast, and real-time responses.

## 🚀Features:
- Voice command recognition using Speech recognition.
- Fetches real-time data using Google API.
- Retrieves summaries and information from Wikipedia API.
- Fast and responsive UI built with React.

## 🛠️Tech Stack:
- React.js
- JavaScript(ES6+)
- HTML5
- CSS3
- Speech Recognition
- Google API
- Wikipedia API
